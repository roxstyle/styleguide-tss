Body Copy

Default font stack: font-family: 'Open Sans', Helvetica, sans-serif coming from google web fonts

Light body text: font-family: 'Open Sans', Helvetica, sans-serif; font-weight 300

Semi-Bold body text: font-family: 'Open Sans', Helvetica, sans-serif; font-weight 600

Bold body text: font-family: 'Open Sans', Helvetica, sans-serif; font-weight 700

Body copy is set in foundation/_functions.scss where the rem-base is set and foundation/_global.scss where body and html are set to base-font-size and line-height:1.35

Paragraph copy is set in foundation/_functions.scss where the rem-base is set and foundation/_type.scss where paragraph tag is set to 1rem and line-height:1.35
