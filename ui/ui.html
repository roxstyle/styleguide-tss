Grid

Basic

Within the outer framework, the content areas are wrapped into sections. The section tag has no margins or padding, and clears everything before and after itself. Within a section, start by adding an element with a class of row. This will create a horizontal block to contain vertical columns. Then add divs with a column class within that row. You can use column or columns - the only difference is grammar. Specify the widths of each column with the small-#, medium-#, and large-# classes.

We are using Foundation Grid which is mobile-first. Code for small screens first, and larger devices will inherit those styles. Customize for larger screens as necessary.

Responsive Gestures

ipad zooming research

We are using the improved ios-viewport-scaling-bug-fix.js at the top of the site script.js. the older version resulted in tabs not working on ipad.

from Mathias Bynens' github


// ios-viewport-scaling-bug-fix
// https://gist.github.com/mathiasbynens/901295
// By @mathias, @cheeaun and @jdalton

(function(doc) {

	var addEvent = 'addEventListener',
		type = 'gesturestart',
		qsa = 'querySelectorAll',
		scales = [1, 1],
		meta = qsa in doc ? doc[qsa]('meta[name=viewport]') : [];

	function fix() {
		meta.content = 'width=device-width,minimum-scale=' + scales[0] + ',maximum-scale=' + scales[1];
		doc.removeEventListener(type, fix, true);
	}

	if ((meta = meta[meta.length - 1]) && addEvent in doc) {
		fix();
		scales = [.25, 1.6];
		doc[addEvent](type, fix, true);
	}

}(document));
