Buttons

As a convention, buttons are used for actions while links are used for objects. For instance, "Download" could be a button and "recent activity" could be a link.

All buttons default to a light gray style, but a number of functional classes can be applied for different color styles. These classes include a .primary class, a .info class, a .success class, and a .inverse class.

Example buttons

Button styles can be applied to anything with the .button applied. Typically you’ll want to apply these to only <a>, <button>, and select <input> elements. Here’s how it looks:
